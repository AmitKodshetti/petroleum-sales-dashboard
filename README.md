# Petroleum Sales & Logistics Dashboard

## Overview

Built an Excel dashboard to analyze petroleum sales, dealer performance, revenue achievement, invoice volume, and on-time delivery using SQL-generated relational business data.

## Tools Used

* MySQL
* Microsoft Excel
* Advanced Excel Formulas
* Data Visualization

## Features

* KPI Cards for Revenue, Target, Achievement %, Invoice Count, and On-Time Delivery
* Dealer Performance Table with Conditional Formatting
* Revenue Comparison Chart
* Dealer Achievement Ranking Chart
